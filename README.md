📌 About This Repository
This repository serves as my personal learning log and project portfolio while studying full-stack web development. It covers both front-end and back-end fundamentals through exercises, mini-projects, and real-world challenges.
Every folder here represents a step forward — no polished final products, just honest progress, iteration, and learning by doing.
